Import Instructions
Supported File Formats:
  • Excel files (.xlsx, .xls)
  • CSV files (.csv)
Required Columns:
  • Title - Project title (required)
  • Principal Investigator - PI name (required)
  • Description - Project description
  • Status - Active, On Hold, Completed, or Cancelled
  • Start Date - Project start date
  • End Date - Project end date
  • Team Members - Comma-separated team member names
  • Funding Source - Funding organization
  • Budget - Project budget amount (can include currency symbols like Rs 50000, USD 25000, or just numbers)
  • Currency - Budget currency (Rs, USD, EUR, INR, GBP, etc. - supports any currency)
Date Format Examples:
  • 2024-12-31
  • 31/12/2024
  • 12/31/2024
  • December 31, 2024
Tips:
  • First row should contain column headers
  • Empty cells will be set as "Not specified"
  • Project IDs will be auto-generated
  • Maximum file size: 10MB
